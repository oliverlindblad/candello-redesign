# Candello Media Website Redesign

Modern, responsive website built with Astro, TailwindCSS, and Motion One.

## Tech Stack

- **Astro** - Static site generator
- **TailwindCSS** - Utility-first CSS framework
- **Motion One** - Animation library

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
├── components/       # Reusable UI components
├── layouts/         # Page layouts
├── pages/           # Astro pages (routes)
├── data/            # Static data (projects, services)
└── styles/          # Global styles and Tailwind config
```

## Pages

- `/` - Forside (Home)
- `/projekter` - Project overview
- `/projekter/[slug]` - Project detail pages
- `/services` - Services
- `/kontakt` - Contact
- `/om-os` - About us
